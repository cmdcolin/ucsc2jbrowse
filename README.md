# ucsc2jbrowse

used for parsing sql and txt.gz database dumps from ucsc and making jbrowse 2
instances

## Pre-requisites

- hck
- fdfind
- node.js
- yarn

## Usage

```bash
./makeAll.sh
```

Downloads mirror to ~/ucsc and builds to ~/ucscResults

## Notes

Loading synteny tracks is a somewhat manual process
